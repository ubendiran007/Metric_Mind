## 🚀 How to Run

### Quick Start
```bash
# Install dependencies (if not already done)
npm install

# Run both frontend and backend together
npm run dev
```

### Access Points
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000

### Alternative (Separate Terminals)
```bash
# Terminal 1 - Frontend
npm start

# Terminal 2 - Backend
npm run server
```
